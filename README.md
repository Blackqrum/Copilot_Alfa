# Альфа-Бизнес Ассистент

AI-ассистент для владельцев малого и среднего бизнеса в России. Предоставляет экспертные консультации по юридическим, маркетинговым, финансовым и управленческим вопросам. Альфа-Бизнес Ассистент - это полнофункциональное веб-приложение, которое сочетает в себе современный React-фронтенд, высокопроизводительный Go-бэкенд и мощную AI-модель Ollama для предоставления интеллектуальных бизнес-консультаций.

## Технологический стек

### Backend
- Go 1.21 - высокопроизводительный серверный язык
- Gin - веб-фреймворк для создания API
- PostgreSQL - реляционная база данных
- Ollama - платформа для работы с AI-моделями

### Frontend
- React 18 - пользовательский интерфейс
- Vite - сборщик и dev-сервер
- CSS3 - стилизация компонентов

### Инфраструктура
- Docker - контейнеризация приложения
- Docker Compose - оркестрация сервисов

## Установка и запуск

### 1. Клонирование репозитория
git clone https://github.com/Blackqrum/Copilot_Alfa.git
cd Copilot_Alfa

### 2. Запуск приложения
docker-compose up --build

### 3. Доступ к приложению
- Frontend приложение: http://localhost:3000
- Backend API: http://localhost:8080

## Первое использование

1. Откройте http://localhost:3000 в браузере
2. Зарегистрируйте нового пользователя
3. Войдите в систему
4. Начните общение с AI-ассистентом в разделе чата
